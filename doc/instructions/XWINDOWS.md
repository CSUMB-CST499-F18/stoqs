Instructions for installing X Window System software on MacOS and Windows
=========================================================================

### MacOS

From OS X Mountain Lion (10.8) Apple dropped dedicated support for the X11.app.
Users are now directed to the open source [XQuartz project](https://www.xquartz.org/).  In order to 
run the STOQS functional tests and to execute Jupyter Notebooks in a STOQS VM
you will need to install XQuartx from [https://www.xquartz.org/](https://www.xquartz.org/).
